# Fastify сервер
Простое веб-приложение на Fastify, объединяющее две контрольные точки:

1. **Первая КТ** – статическая страница с кнопкой и API.
2. **Вторая КТ** – динамические страницы на Pug для управления списком пользователей (просмотр, добавление, удаление).

## Как запустить

### 1. Клонирование репозитория
```bash
git clone https://github.com/Promathpromath/server_kt.git
cd server_kt
```

## Установка зависимостей
Убедитесь, что установлен Node.js (версия 18 или выше). Затем выполните:

```bash
npm install
```
Будут установлены fastify, @fastify/static, @fastify/view, @fastify/formbody и pug.


Запуск сервера
```bash
npm start
```


После запуска в терминале появится сообщение:
Сервер запущен на http://localhost:3000




# Вторая контрольная точка (управление пользователями)

### Что добавлено
Шаблонизатор Pug для генерации HTML.

Массив пользователей в памяти сервера (id, name, email).

Единая страница по адресу /, которая включает:

блок с кнопкой API (из первой КТ),

список всех пользователей,

кнопку удаления для каждого пользователя,

сворачиваемую форму добавления нового пользователя.

Обработка отправки формы (POST /users) и удаления (POST /users/delete/:id).

### Как протестировать работу с пользователями
Запустите сервер (npm start) и откройте http://localhost:3000.

В правой карточке вы увидите список пользователей (изначально два).

Нажмите «Добавить пользователя» – откроется форма.

Введите имя и email, нажмите «Создать».

Новый пользователь появится в списке.

Для удаления нажмите кнопку «Удалить» рядом с нужной записью – пользователь исчезнет.

Все изменения происходят без перезагрузки сервера, данные хранятся в памяти (при перезапуске сервера возвращаются к исходным двум пользователям).
