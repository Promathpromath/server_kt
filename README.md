# Fastify Server (server_kt1)

Простой сервер на микрофреймворке Fastify с веб-интерфейсом и API-эндпоинтом.

## Функциональность

- Главная страница (`/`) с красивой кнопкой.
- При нажатии на кнопку отправляется GET-запрос на `/api`.
- Сервер отвечает JSON: `{ "message": "Запрос прошел успешно" }`.
- В консоли браузера выводится полученное сообщение.
- На странице отображается статус запроса и есть ссылка на исходный код.

## Запуск сервера

### 1. Клонирование репозитория

Используйте HTTPS (рекомендуется, если не настроен SSH):

 
git clone https://github.com/Alenka319/server_kt1.git
cd server_kt1


## 2. Установка зависимостей
Убедитесь, что у вас установлен Node.js (версия 18 или выше). Затем выполните:
 
npm install
Эта команда установит fastify и @fastify/static — все необходимые модули.

## 3. Запуск сервера
bash
npm start
После запуска в терминале появится сообщение:
Сервер запущен на http://localhost:3000

## 4. Проверка работы
Откройте браузер и перейдите по адресу http://localhost:3000.

Нажмите кнопку «Сделать запрос на АПИ».

В консоли браузера (F12 → вкладка Console) должно появиться сообщение «Запрос прошел успешно».

На странице также отобразится статус запроса.
