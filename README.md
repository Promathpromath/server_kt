# Fastify сервер

Простое веб-приложение на Fastify с одной кнопкой и API.

## Как запустить

### 1. Установите зависимости
Откройте терминал в папке проекта и выполните:


### 1. Клонирование репозитория

Используйте HTTPS (рекомендуется, если не настроен SSH):

git clone https://github.com/Promathpromath/server_kt.git

cd server_kt1


## 2. Установка зависимостей
Убедитесь, что у вас установлен Node.js (версия 18 или выше). Затем выполните:
 
npm install
Эта команда установит fastify и @fastify/static — все необходимые модули.

## 3. Запуск сервера

npm start

После запуска в терминале появится сообщение:
Сервер запущен на http://localhost:3000

Откройте браузер и перейдите по адресу:
[http://localhost:3000](http://localhost:3000)

На странице нажмите кнопку «Сделать запрос на АПИ».  
В консоли браузера (F12 → Console) должно появиться сообщение:  
**«Запрос прошел успешно»**

## Файлы проекта
- `server.js` – код сервера.
- `public/index.html` – страница с кнопкой.
- `package.json` – список зависимостей.
